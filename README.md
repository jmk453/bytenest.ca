# Requirements

- Brotli
- Rsync
- Python

# Instructions

You can view your local copy at [localhost:8000](https://localhost:8000):

```console
$ ./serve
```

You can publish your local copy to [bytenest.ca](https://bytenest.ca):

```console
$ ./publish
```
